# EDINETコード対応表

`EdinetcodeDlInfo.csv` をここに配置してください。

## ダウンロード方法

1. https://disclosure2.edinet-fsa.go.jp/weee0010.aspx にアクセス
2. 「EDINETコードリスト」の「ダウンロード」をクリック
3. ZIPを展開し、`EdinetcodeDlInfo.csv` をこのディレクトリに配置

## ファイル形式

- エンコーディング: CP932 (Shift_JIS)
- 列: EDINETコード, 提出者種別, 上場区分, 連結の有無, 資本金, 決算日, 提出者名, ...
