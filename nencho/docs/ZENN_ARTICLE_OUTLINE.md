# Zenn 記事 構成案・要件メモ

**参考**: [確定申告を自動化する Agent Skill "shinkoku" を OSS にした](https://zenn.dev/kazukinagata/articles/83fe82191db01b)

## 記事の位置づけ

- **読者**: 個人事業主・経理担当・エンジニアで「年末調整を楽にしたい」「AI エージェントで業務を効率化したい」人
- **メッセージ**: 確定申告の shinkoku と同じ設計思想で、**年末調整**をスキル化して OSS にした；エージェントと対話しながら計算〜源泉徴収票出力まで進められる
- **トーン**: 実用的・誠実（免責を明示しつつ、できること／できないことをはっきり書く）

## 見出し構成案

### 1. タイトル（案）

- 「年末調整をエージェントと一緒にやる Agent Skill「nencho」を OSS にした」
- または「確定申告の次は年末調整——nencho をスキル化して OSS にした」

### 2. 導入（〜200字）

- 年末調整の「面倒さ」（年に1回で忘れる、控除の種類が多い、源泉徴収票の作成など）
- 「確定申告は shinkoku で自動化されている。年末調整も同じようにエージェントに任せたい」→ nencho をスキル化して OSS にした、という流れ

### 3. 作った動機

- なぜ年末調整に着目したか（給与所得者・小規模法人の実務で毎年発生する負荷）
- shinkoku の設計（スキル分離・SKILL.md 標準）をそのまま年末調整に適用できると考えた、という一貫性

### 4. OSS にした動機（任意）

- エンジニア以外にも「エージェントでここまでできる」を体感してほしい
- 経理・人事の実務改善のきっかけにしたい

### 5. nencho でできること

- 書類の整理・入力から計算・源泉徴収票出力までの流れを、エージェントとの対話で進められる
- **メインワークフロー（表）**: setup / calculate / validate / export の4スキル
- **補助スキル（表）**: employees（従業員・扶養・一括取込）、capabilities（対応範囲・制限）
- 計算は API と CLI の両方から利用可能であること（スキルがその手順を案内）

### 6. 対象ユーザー

- README の「想定ユーザー」表を要約（給与のみ・扶養・社会保険・生命・地震・住宅ローン・所得金額調整控除・中途入退社 など）

### 7. 非対応項目

- README の「非対応」表を要約（退職所得・e-Tax 自動入力 など）
- 分離課税・事業所得は確定申告（shinkoku）側の領域であることを一言

### 8. 対応エージェント

- SKILL.md オープン標準に準拠しているため、Cursor / Claude Code / Windsurf / GitHub Copilot などスキル対応エージェントで利用可能
- スキルのインストール方法: `npx skills add Majiro-ns/nencho` または clone して API/CLI を直接利用

### 9. セットアップ

- 前提: Python 3.10+、pip
- 手順: clone → `pip install -r requirements.txt` → API 起動（`uvicorn api.main:app`）または CLI（`python -m src.cli.nencho_cli`）
- 作業ディレクトリに DB ができること、個人情報を含むので .gitignore を推奨すること

### 10. テスト・品質への気の配り（任意）

- テストスイートの存在（控除・税額・端数・中途入退社・バリデーションなど）
- 法令・端数処理の根拠（国税通則法・所得税法・令和7年分の表）をコード内で参照していること
- 実データでの検証（freee 等との突合）は今後の課題として触れてもよい

### 11. 実務上の注意・今後の予定

- 令和7年分を正式対応としていること（令和8年分は税制次第で対応予定）
- 計算結果・源泉徴収票は提出前に必ず本人確認すること（免責の再掲）
- 今後の拡張: 令和8年分対応、バリデーション項目の拡充、サンプルデータの整備 など（不足点ドキュメントと連動）

### 12. 最後に

- 自分自身や自社で使ってみて、OSS として公開した経緯
- フィードバック・コントリビュートを歓迎する旨
- リポジトリ URL と README への誘導

### 13. 参考資料

- shinkoku
- SKILL.md オープン標準（または Cursor Skills ドキュメント）
