## パイプライン構成（2026-02-28 更新済み）

### 日次スケジュール（crontab 登録済み）

| 時刻 | Stage | コマンド | 内容 |
|------|-------|---------|------|
| 07:00 | Stage 1 | `stage1_keirin` | Kドリームス取得 → フィルター → リクエストYAML生成 |
| 09:00 | Stage 2 | `stage2_haiku_keirin` | Haiku API で予測テキスト自動生成 |
| 18:00 | Stage 3 | `stage3_keirin` | 結果集約 → bet計算 → Discord通知 |

### フィルター設定（filters.yaml 更新済み）

- **race_type**: 特選 / 二次予選 / 準決勝 / 選抜（追加: 準決勝・選抜）
- **min_confidence_score**: 1（新規追加）
- **Type分類**: expected_payout未設定時はconfidence_scoreで代替
  - conf=0 → C（堅実）
  - conf=1 → A（標準）
  - conf≥2 → B（穴狙い）

## Go/No-Go 判定基準（3/7 週次レビュー時）

### Go（本格稼働継続）
- [ ] Stage 1→3 自動完走率 ≥ 80%（5/7日以上）
- [ ] 1日平均BUYレース数 ≥ 3件
- [ ] Type A または B の的中が1件以上

### 要調整（フィルター見直し）
- [ ] BUYレース数が1件/日未満 → race_type または min_confidence_score を緩和
- [ ] 全件外れ → フィルター過剰か予測品質の問題を特定
- [ ] Stage 2 Haiku エラー多発 → 手動 Stage 2 に一時切替
